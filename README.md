# students-data-app

ccc 151 project

developed using pyqt6

setting up:

download or clone this repo, then

1. create environment
```python -m venv env```

2. activate environment
- Linux/macOS : ```source env/bin/activate```
- Windows : ```env\Scripts\activate.bat```

3. install pyqt6
```pip install pyqt6```

to run :

run main.py
or 
```python main.py```
or
```python -m main```
